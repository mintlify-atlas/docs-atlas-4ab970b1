# Repomix Runner VSCode Extension

A community-maintained VSCode extension called [Repomix Runner](https://marketplace.visualstudio.com/items?itemName=DorianMassoulier.repomix-runner) (created by [massdo](https://github.com/massdo)) lets you run Repomix right inside your editor with just a few clicks.

## Installation

<Steps>

<Step title="Open VSCode">

Launch Visual Studio Code.

</Step>

<Step title="Open Extensions">

Click the Extensions icon in the Activity Bar or press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS).

</Step>

<Step title="Search and Install">

Search for "Repomix Runner" and click Install.

Alternatively, visit the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=DorianMassoulier.repomix-runner) directly.

</Step>

</Steps>

## Features

- **Run on any folder**: Execute Repomix on any folder in your workspace
- **Manage outputs**: Control everything through VSCode's intuitive interface
- **Output options**: Choose between file output or just the content
- **Automatic cleanup**: Optional automatic cleanup of generated files
- **Configuration support**: Works smoothly with your existing `repomix.config.json`

## Usage

The Repomix Runner extension integrates seamlessly into VSCode's interface:

1. Right-click on any folder in the Explorer
2. Select "Run Repomix" from the context menu
3. Choose your preferred options
4. View or save the generated output

## Configuration

The extension respects your existing Repomix configuration:

- Uses `repomix.config.json` if present in your project
- Supports all standard Repomix configuration options
- Allows per-run customization through the VSCode interface

## Benefits

- **Integrated workflow**: No need to switch to terminal
- **Visual interface**: Easy-to-use GUI for all options
- **Quick access**: Run Repomix with just a few clicks
- **Flexible output**: Choose how and where to save results
