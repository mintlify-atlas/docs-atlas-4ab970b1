# Browser Extension

Get instant access to Repomix directly from any GitHub repository! The Repomix browser extension adds a convenient "Repomix" button to GitHub repository pages.

![Repomix Browser Extension](https://raw.githubusercontent.com/yamadashy/repomix/main/website/client/src/public/images/docs/browser-extension.png)

## Installation

### Chrome

Install from the Chrome Web Store:

[Repomix - Chrome Web Store](https://chromewebstore.google.com/detail/repomix/fimfamikepjgchehkohedilpdigcpkoa)

### Firefox

Install from Firefox Add-ons:

[Repomix - Firefox Add-ons](https://addons.mozilla.org/firefox/addon/repomix/)

## Features

- **One-click access**: Pack any GitHub repository directly from the repository page
- **Seamless integration**: Works directly within GitHub's interface
- **More features coming soon**: The extension is actively being developed

## How to Use

<Steps>

<Step title="Navigate to a GitHub repository">

Visit any GitHub repository you want to analyze.

</Step>

<Step title="Click the Repomix button">

Look for the Repomix button added to the GitHub interface and click it.

</Step>

<Step title="Configure options">

Select your preferred output format and options for packing the repository.

</Step>

<Step title="Generate output">

The extension will pack the repository using Repomix and provide you with the output.

</Step>

</Steps>

## Benefits

- **No installation required**: Use Repomix without installing the CLI tool
- **Quick analysis**: Instantly analyze any public GitHub repository
- **Convenient workflow**: Stay in your browser while working with repositories
