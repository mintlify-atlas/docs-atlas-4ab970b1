  <Steps>
    <Step title="Install Repomix">
      Run Repomix instantly with npx, or install it globally:
      
      <CodeGroup>
      ```bash npx
      npx repomix@latest
      ```
      
      ```bash npm
      npm install -g repomix
      ```
      
      ```bash yarn
      yarn global add repomix
      ```
      
      ```bash bun
      bun add -g repomix
      ```
      
      ```bash Homebrew
      brew install repomix
      ```
      </CodeGroup>
    </Step>

    <Step title="Run in your project">
      Navigate to your project directory and run Repomix:
      
      ```bash
      repomix
      ```
      
      This generates a `repomix-output.xml` file containing your entire repository in an AI-friendly format.
    </Step>

    <Step title="Use with AI tools">
      Upload the generated file to your favorite AI assistant:
      
      <Note>
      Send the file to Claude, ChatGPT, or any LLM with a prompt like: "This file contains my entire codebase. Please review the architecture and suggest improvements."
      </Note>
    </Step>
  </Steps>
</div>
